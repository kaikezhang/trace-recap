# TraceRecap iOS App — 产品功能描述

> 供 UI/UX 设计师参考 · 纯功能与交互逻辑 · 不限定视觉方案

---

## 产品一句话

**让普通旅行者在 3 分钟内把旅行路线变成精美动画短视频。**

## 目标用户

旅行回来想发朋友圈 / Instagram / 抖音 / 小红书的普通人。  
不是视频剪辑师，不想学复杂工具，只想**快速、好看、能分享**。

## 竞品参考

- **TravelBoast** — 可爱卡通图标 + 地图动画（最接近的竞品）
- **Polarsteps** — 旅行轨迹记录（太重，不是短视频工具）
- **剪映 / CapCut** — 通用视频编辑（太复杂，不够聚焦）

---

## 用户核心流程

```
打开 App → 新建项目 → 添加城市 → 选交通方式 → 加照片 → 预览动画 → 导出视频 → 分享
```

全程 < 3 分钟。

---

## 一、首页 / 项目列表

用户打开 App 看到的第一个界面。

**能做什么：**
- 查看所有已创建的项目
- 每个项目显示：名称、路线缩略图、城市数量、创建日期
- 新建项目
- 导入别人分享的路线文件（JSON）
- 长按项目：重命名 / 复制 / 删除

**首次打开（无项目时）：**
- 空状态引导用户创建第一个项目

---

## 二、编辑器

编辑器是 App 的核心界面。用户在这里完成所有操作：添加城市、选交通方式、加照片、预览、导出。

### 2.1 地图区域

编辑器的主体是一张 **2D 平面地图**。

**地图上显示：**
- 每个目的地的标记点
- 目的地之间的路线连线（不同交通方式颜色和线型不同）
- 播放动画时的交通工具图标和城市名标签

**3 种地图风格可切换：**
- 浅色（简洁白底）
- 深色（暗黑高级感）
- 卫星（真实影像）

**手势：**
- 双指缩放、单指拖拽浏览地图

### 2.2 添加目的地

用户有两种方式添加城市：

**搜索添加：**
- 搜索框输入城市名（支持中英文）
- 显示搜索建议列表
- 点击某个结果 → 城市添加到路线末尾，地图上出现标记

**地图点击添加：**
- 在地图上长按某处 → 自动获取地名 → 添加到路线

**AI 生成路线（增强功能）：**
- 用户输入一段文字，如 "从北京飞东京，新干线去京都，再坐火车去大阪"
- AI 自动解析生成完整路线
- 生成后用户可手动修改

### 2.3 路线列表

以列表形式展示所有目的地，按顺序排列。

**每个目的地显示：**
- 序号
- 城市名称
- 照片缩略图和数量（如有）
- 途经点标识（如已标记）

**两个目的地之间显示：**
- 当前选的交通方式图标
- 这段路线的播放时长

**操作：**
- 拖拽调整目的地顺序
- 左滑或长按删除某个目的地
- 点击目的地 → 进入照片管理 / 排版编辑

### 2.4 选择交通方式

每两个相邻目的地之间需要选一种交通方式。

**7 种可选：**
1. ✈️ 飞机
2. 🚗 汽车
3. 🚆 火车
4. 🚌 巴士
5. ⛴️ 轮渡
6. 🚶 步行
7. 🚲 自行车

点击交通方式区域 → 弹出选择器 → 选完后路线自动更新。

不同交通方式在地图上的路线线型不同（如飞机是弧线虚线，汽车是实线沿公路走）。

### 2.5 途经点

有些城市只是路过，不是真正停留。标记为"途经点"后：
- 动画播放时摄像机飞过这个城市不停留
- 不显示城市名标签
- 不显示照片
- 路线连贯不中断

**举例：**
北京 → **郑州**（途经）→ 武汉  
动画中摄像机从北京一路飞到武汉，经过郑州但不暂停。

**操作：**
- 在目的地卡片上有开关，可切换"停靠站 / 途经点"

### 2.6 照片

每个目的地最多添加 **9 张照片**。

**添加方式：**
- 从系统相册选择（支持批量选）
- 拍照添加

**照片操作：**
- 删除
- 添加文字说明（caption）
- 调整裁剪焦点（拖拽确定照片居中位置，影响裁剪时保留哪个区域）
- 拖拽调整照片顺序

**照片在动画中的表现：**
- 摄像机到达某个有照片的城市时，地图暂停
- 照片以卡片形式优雅出现在画面上
- 停留约 2 秒
- 然后以动画收起（缩小 + 飘走），继续下一段路线

### 2.7 照片排版

照片在画面上的排列方式。

**自动排版（默认）：**
系统根据照片数量自动选择最佳排列。比如：
- 1 张 → 单张居中大图
- 2 张 → 并排
- 3 张 → 1 大 + 2 小
- 4 张 → 2×2 网格
- 9 张 → 3×3 九宫格

**手动排版：**
用户可选择 5 种排版模板：
1. **网格** — 均匀方格排列
2. **主图** — 一张大 + 其余小
3. **瀑布流** — 高低错落
4. **胶片** — 横向电影胶片条
5. **散落** — 随机角度散落，像桌上的实体照片

**可调参数：**
- 照片顺序（拖拽调整）
- 间距大小
- 圆角大小

**编辑交互：**
- 用户点击某个目的地的照片 → 进入排版编辑
- 地图自动跳转到该城市并显示照片预览效果
- 所见即所得，调整实时反映

### 2.8 段落时间

每段路线的动画时长。

**默认：自动计算。** 系统根据总城市数和路线距离自动分配，用户不需要管。

**手动调整（可选）：**
- 用户可以给任意一段路线设定自定义时长（2~30 秒）
- 适合觉得某段太快或太慢的情况
- 可随时恢复为自动
- 一键清除所有自定义时长

### 2.9 城市名标签

动画播放时，到达和离开城市会显示城市名标签。

**可调：**
- 语言：英文 / 中文
- 字号：小 ~ 大（可调滑块）

### 2.10 播放预览

在编辑器内直接预览完整动画效果。

**控件：**
- ▶️ 播放 / ⏸️ 暂停
- ⏮️ 重置回起点
- 进度条：可拖拽跳转到任意时刻
- 当前时间 / 总时长显示

**播放时看到什么：**
- 摄像机在城市间平滑飞行
- 卡通交通工具图标沿路线移动（会根据方向变换朝向）
- 路线随图标移动逐渐"画出来"
- 到达城市时弹出城市名标签
- 有照片的城市会暂停展示照片卡片
- 照片收起后继续下一段

### 2.11 视频导出

**导出前选择：**
- 画面比例：横屏（16:9）或竖屏（9:16，适合 Stories/抖音）
- 分辨率：720p 或 1080p

**导出过程：**
- 显示进度条
- 支持取消

**导出后：**
- 保存到系统相册
- 直接分享到其他 App（微信、Instagram、抖音等，调用系统分享面板）

### 2.12 路线存取

**保存路线：** 将当前路线导出为文件，可分享给朋友。  
**导入路线：** 打开别人分享的路线文件，一键加载所有城市和交通方式。

## 三、动画效果描述

> 设计师需要理解动画节奏以设计对应的 UI 状态

每段路线的动画分 5 个阶段：

| 阶段 | 用户看到什么 |
|------|------------|
| **停留（出发）** | 摄像机停在出发城市上方，显示城市名 |
| **拉远** | 摄像机缓缓拉远，让出发城市和目的城市同时出现在画面中 |
| **飞行** | 卡通交通工具图标沿路线移动，路线逐渐画出来 |
| **推近** | 摄像机平滑推近目的城市 |
| **到达** | 停在目的城市上方，弹出城市名。如有照片则展示照片卡片 |

然后无缝进入下一段。

**照片退场效果：**
- 照片不是瞬间消失，而是逐张依次收起
- 每张缩小 + 向上飘走 + 渐隐
- 在下一段开始飞行前完成退场

## 四、关键用户场景

### 场景 A：第一次使用
新用户，没有项目。
→ 空状态引导 → 创建项目 → 搜索添加 2 个城市 → 选交通方式 → 播放预览 → 导出

### 场景 B：完整旅行回忆
用户刚从台湾旅行回来。
1. 新建项目"台湾环岛"
2. 添加 6 个城市：台北、台中、台南、嘉义、阿里山、台北
3. 选交通方式：大部分火车，嘉义→阿里山选巴士
4. 给台北加 3 张照片（101、夜市、淡水）
5. 给阿里山加 2 张照片（日出、小火车）
6. 预览，觉得阿里山段太快 → 手动调到 8 秒
7. 切中文标签，字号调大
8. 选竖屏 9:16，导出 1080p
9. 保存到相册 → 发朋友圈

### 场景 C：朋友分享
朋友发了一个路线文件过来。
→ 导入文件 → 路线自动加载 → 替换成自己的照片 → 导出

### 场景 D：复杂路线
10 个城市的欧洲行，其中 3 个只是转车经过。
→ 把 3 个转车城市标记为途经点 → 动画流畅飞过不停留 → 只在 7 个真正停留的城市展示照片

## 五、MVP 范围

### ✅ 做
- 项目管理（创建 / 保存 / 导入 / 导出）
- 地图 + 3 种风格
- 添加目的地（搜索 + 地图点击）
- 7 种交通方式
- 途经点
- 照片（每站最多 9 张）
- 照片排版（自动 + 5 种手动模板）
- 段落时间控制（自动 + 手动）
- 城市名标签（中英文、可调字号）
- 动画预览（播放/暂停/进度条）
- 视频导出（横屏/竖屏，720p/1080p）
- 分享（系统分享面板）

### ❌ 不做
- 用户账号 / 登录
- 云同步
- 付费 / 订阅
- 社区画廊
- BGM 背景音乐
- 片头片尾
- 文字水印
- 自定义地图配色
- GPX / KML 轨迹导入
- 3D 地球视图
- 时间轴编辑器
