# Spec: TraceRecap UI/UX/Landing Page Polish — Production Quality

## Goal
Elevate TraceRecap from functional MVP to industry-leading quality. The product should feel like it was designed by a top-tier design team — not generated by AI or from a template. Think Linear, Framer, Raycast level of craft.

## Current State Assessment

### Landing Page (`src/app/page.tsx`)
- **Generic SaaS template feel** — gradient text, symmetrical 3-card grid, placeholder-style hero mockup
- Hero section is an SVG mockup, not actual product footage
- "How it works" section is standard 3-column icon cards — forgettable
- No social proof (no users, testimonials, numbers)
- No real product demo or video showcase
- Footer is a single line — no links, no personality
- Missing: FAQ, use cases, competitive differentiation, feature showcase
- gradient-mesh background is subtle but generic

### Editor UI (`src/components/editor/`)
- **Functional but dense** — lots of options crammed into toolbar
- TopToolbar has too many controls, feels cluttered on smaller screens
- LeftPanel is basic — just search + route list, no visual hierarchy
- Settings panel (dropdown) could be more intuitive
- No empty state illustrations or personality
- Playback controls are functional but not polished
- Export dialog could be more engaging
- No dark mode for editor (only light)
- Missing: keyboard shortcuts panel, quick tips, smooth transitions between states

### Global
- Only Geist font family — clean but generic
- Color palette is default shadcn (gray/black) — no brand identity
- No favicon or OpenGraph/social preview images
- No loading animations or skeleton screens
- Inconsistent spacing and visual rhythm

## Design Direction

### Brand Identity
- **Warm, travel-inspired palette** — sunset oranges, ocean blues, earthy greens, not tech-startup purple
- **Typography**: Keep Geist Sans for UI, add a display/heading font with character (e.g., General Sans, Cabinet Grotesk, or Satoshi)
- **Visual language**: Organic shapes, map-inspired elements, subtle paper/travel textures
- **Personality**: "Your personal filmmaker for travel memories" — warm, approachable, slightly playful

### Landing Page Redesign
1. **Hero Section**
   - Story-driven headline with emotional appeal, not feature-centric
   - Embedded product demo (autoplay video loop of an actual TraceRecap animation, or interactive mini-demo)
   - Asymmetric layout — NOT centered text + centered image
   - Floating travel elements (stamps, tickets, pins) for personality
   - Clear primary CTA: "Create Your First Recap" + secondary: "Watch Demo"

2. **Social Proof Bar**
   - "X routes animated" counter (can be generated from real data or estimated)
   - Logos or badges (Mapbox, Next.js, WebCodecs)
   - "Made for Instagram, TikTok, WeChat Moments" with platform icons

3. **Feature Showcase** (replace generic 3-cards)
   - Interactive feature sections that respond to scroll/hover
   - Show actual product screenshots or mini-videos for each feature cluster:
     - Route Building (search, drag, transport modes)
     - Photo & Video (cards, layouts, albums)
     - Export (one-click, multiple formats)
   - Staggered layout, not a grid

4. **Use Cases / Gallery**
   - Show 3-4 real example outputs (GIF/video previews of animated routes)
   - Different trip types: Europe backpacking, Asia road trip, US cross-country, city walking tour

5. **How It Works** (reimagined)
   - Numbered steps with inline product screenshots
   - Progress line connecting steps (like a route on a map!)
   - 3 steps max, bigger visual weight

6. **FAQ Section**
   - Collapsible, clean, addresses key concerns (cost, browser support, data privacy)

7. **Footer**
   - GitHub link, contact, social media
   - Product Hunt / upcoming badges
   - "Built with ❤️ for travelers" with more personality

### Editor UX Polish
1. **TopToolbar Redesign**
   - Simplify: group related controls, use icon-only with tooltips
   - Breadcrumb: Home > Project Name > Editor
   - Save status indicator (cleaner, less intrusive)
   - Move settings into a dedicated sidebar/panel instead of dropdown

2. **Left Panel Enhancement**
   - Visual route summary at top (mini-map or distance/time)
   - Better location cards with photo thumbnails
   - Clearer drag handles and reorder feedback
   - Section headers for route vs settings

3. **Empty States**
   - Custom illustration or animation for "no cities added yet"
   - Guided first-action prompt

4. **Transitions & Micro-interactions**
   - Smooth panel open/close with framer-motion
   - Button press feedback
   - Loading skeletons for geocoding results
   - Toast notifications for save/export events

5. **Responsive Polish**
   - Bottom sheet improvements for mobile
   - Touch-friendly controls
   - Proper mobile navigation
